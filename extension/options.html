  • Lets you control video and audio players in websites using the Media Controller plasmoid.

  • Adds a context menu entry to links enabling you to send them to your phone and other paired devices using KDE Connect.

  • Make sure the “Browser Tabs” module is enabled in Plasma Search settings.

  • Tracks visited websites in KDE Activities and also allows for easy sharing using the Quick Share plasmoid.

  • Adds an icon to System Tray whenever an incognito window is opened that allows to close them at once.
    This only works if the extension is allowed to run in Incognito mode at which point incongito pages are available to the listed features (including KRunner, if enabled).
